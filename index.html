About Me

Web-developer

I am Web Developer/Designer

I use editors and software: SublimeText, VS Code, Chrome Dev Tools, OpenServer, Photoshop, Avocode, Figma.

Since my specialization is an HTML layout designer, I have a good command of HTML5 and CSS3 (I use Pug, SASS preprocessors), I also use the Git version control system. In my work I adhere to the principles of cross-browser and adaptive (grid, flexbox), semantic and accessible ( Accessibility) layouts. During the execution of orders, I used the Bootstrap CSS framework.

I have experience with CMS WordPress, my task was to adapt / integrate the layout from .psd, .fig to this CMS.

Almost no site can do without JavaScript, so I use jQuery to add galleries and lightboxes to the page with the connection of the corresponding plugins of this library and their configuration. To optimize the code and speed up the layout process, I use the Gulp task manager.

Layout of HTML templates according to source .psd and .fig layouts.

Constantly learning new technologies and trying to use them in projects.
